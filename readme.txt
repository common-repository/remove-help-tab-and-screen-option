== Description ==
This plugin removes the screen option and help tabs depending on this plugin settings. By default this plugin removes both.

* By default this plugin remove Help and Screen options tab
* For enable any option go to this plugin setting.

== Installation ==

This section describes how to install the plugin and get it working.

In WordPress:

    Go to Plugins > Add New > search for :: Remove Help tab and Screen option ::
    Press Install Now for the Remove Help tab and Screen option
    Press Activate Plugin
		
To install manually instead:

    Upload the remove-screen-help directory to the /wp-content/plugins/ directory
    Activate the plugin through the Plugins menu in WordPress
